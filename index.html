Whats so special about frag.jetzt?

Differnces to other student-teacher communication tools

Anonymous

Questions are asked that nobody dares to ask in the large lecture hall

Ranking System

Students can upvote questions. Highest voted questions are displayed on top

Easy Moderation

The question list can easily be moderated live by an assistant. Thus there is no abuse

Open Source

frag.jetzt is an Open Source tool and therefore free of charge
